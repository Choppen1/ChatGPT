# Dreamy Flappy Bird

Een moderne en stijlvolle interpretatie van Flappy Bird met vloeiende animaties,
parallax achtergronden en prachtige kleurenovergangen. Alles wordt
proceduraal getekend met pygame – geen spritesheets nodig.

## Installatie

1. Zorg dat je Python 3.9 of hoger geïnstalleerd hebt.
2. Installeer de afhankelijkheid:

   ```bash
   pip install -r requirements.txt
   ```

## Spelen

Start het spel met:

```bash
python flappy_birds.py
```

### Besturing

- **SPACE**, **pijl omhoog** of **W**: flap
- **Muisklik**: flap
- **ESC** of **Q**: sluiten

## Highlights

- Dag- en nachtcyclus met zachte kleurverlopen
- Meerdere parallaxlagen (wolken, bergen, heuvels en grond)
- Expressieve vogel met geanimeerde vleugel, rotatie en idle animatie
- Aanpasbare moeilijkheid: de gap wordt kleiner naarmate je verder komt
- Highscore wordt automatisch lokaal opgeslagen
